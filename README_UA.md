# Клініка Asklepiy — Медична Архітектура 2026

> Продакшн-рівень, ультра-преміум, mobile-first веб-екосистема для провідної медичної клініки України "Asklepiy".

## Огляд проекту

Цифрова платформа клініки Asklepiy — це медичний веб-досвід нового покоління, що поєднує:

- **Точність** — стерильний, мінімалістичний дизайн, що передає медичний авторитет
- **Довіра** — прозорі ціни, реальні профілі лікарів, професійна естетика
- **Технологія** — AI-асистент, розумна система запису, цифровий кабінет пацієнта
- **Продуктивність** — цільовий показник Lighthouse 100 за всіма категоріями

## Ключові функції

### AI-асистент Asklepiy
- Плаваючий glassmorphism-віджет (нижній правий кут)
- Контекстно-залежні повідомлення залежно від сторінки та часу перебування
- Семантичне розпізнавання симптомів із рекомендаціями відділень
- Підтримка двох мов (Українська / Англійська)

### Розумна система запису (4-крокова)
- Вибір послуги з візуальними плитками категорій
- Вибір лікаря з рейтингами та доступністю
- Анімований кастомний календар + розумні пропозиції часу
- Форма підтвердження з підсумком запису

### Кабінет пацієнта
- Симуляція аутентифікації (Email, SMS, Google OAuth)
- Перемикач сімейного профілю (Я / Дитина / Партнер)
- Записи, результати досліджень з AI-інтерпретацією
- Інтерактивна хронологія здоров'я
- Сховище документів

### Локалізація
- Українська (за замовчуванням) + Англійська
- Миттєве перемикання без перезавантаження (next-intl v4)

### Mobile-First UX
- Повноекранне glassmorphism мобільне меню
- Оптимізований для сенсорного екрана календар запису
- Адаптивність для всіх розмірів екранів

```
│       ├── napryamky/        # Напрямки
│       ├── likari/[slug]/    # Лікарі + профілі
│       ├── tsiny/            # Ціни
│       ├── novyny/           # Новини
│       ├── analizy/          # Аналізи
│       ├── deklaratsiya/     # Декларація
│       ├── pro-nas/          # Про нас
│       ├── kontakty/         # Контакти + карта
│       ├── sign-in/          # Аутентифікація
│       └── dashboard/        # Кабінет пацієнта
├── components/
│   ├── ui/                   # Дизайн-система (Button, Card, Badge, Input, Skeleton)
│   ├── layout/               # Header, Footer, MobileMenu
│   ├── sections/             # Секції головної сторінки
│   └── providers/            # React провайдери
├── features/
│   ├── ai-assistant/         # AI-віджет
│   ├── booking/              # Майстер запису
│   ├── doctors/              # Компоненти лікарів
│   └── services/             # Функція послуг
├── i18n/                     # Конфігурація next-intl
├── lib/
│   ├── api/                  # Шар mock-API
│   └── utils/                # Утиліти
└── middleware.ts              # Middleware для i18n
```

## Дизайн-система — "Medical Architecture 2026"

### Кольорова палітра
```
Основний:   #0D3A7E  (Глибокий медичний синій)
Акцент:     #1A9EC9  (Медичний блакитний)
Поверхня:   #FFFFFF  (Чисто білий)
Фон:        #F2F6FB  (Світло-блакитно-сірий)
Текст:      #0C1929  (Темно-синій)
Приглушений: #4A6180 (Сталевий синій)
```

### Типографіка
- Основний: Inter (300, 400, 500, 600)
- Легкі ваги для преміум відчуття

### Принципи дизайну
- `rounded-[6px]` — ніякого "бульбашкового" UI
- Glassmorphism на хедері, мобільному меню, AI-віджеті
- "Розкішний пробіл" (7rem відступ секцій)
- Framer Motion для всіх змін стану

## Початок роботи

### Передумови
- Node.js 18+
- npm 9+

### Встановлення

```bash
git clone https://github.com/your-org/asklepiy-clinic.git
cd asklepiy-clinic
npm install
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000) — перенаправляє на `/uk`.

### Змінні середовища

Створіть `.env.local`:
```env
NEXT_PUBLIC_MAPS_API_KEY=ваш_ключ_google_maps
```

## Тестування

```bash
npm run test            # Юніт-тести
npm run test:coverage   # З покриттям
npm run test:e2e        # E2E тести (потрібен dev-сервер)
npm run test:e2e:ui     # E2E з інтерфейсом Playwright
```
